# 📌 Project Overview
This project focuses on performing **Exploratory Data Analysis (EDA)** using SQL on the **Gold Layer** of a Data Warehouse project.  
The objective of this project is to explore business data, analyze trends, discover insights, and practice analytical thinking using SQL queries.

The EDA process was structured into **6 analytical stages**, and all SQL scripts were version-controlled and committed using Git 🛠️.

This project is built on top of my previously developed **Data Warehouse Project**, where the data was cleaned, transformed, and modeled before analysis , Here is link to my datawarehouse project : https://github.com/mahdddadg/sql-data-warehouse

# 🗂️ Project Structure

The EDA project is divided into the following stages:

## 1️⃣ Database Exploration
- Exploring database structure
- Understanding tables and relationships
- Inspecting schemas and columns
- Validating available data for analysis

### 🔍 Topics Covered
- Database inspection
- Metadata analysis
- Schema understanding

---

## 2️⃣ Dimension Exploration
Analyzing descriptive business dimensions to better understand entities within the dataset.

### 📌 Examples
- Customers
- Products
- Categories
- Countries
- Cities

### 🧠 SQL Concepts Used
- DISTINCT
- GROUP BY
- ORDER BY
- Filtering and sorting

---

## 3️⃣ Measure Exploration
Exploring quantitative business metrics and numerical values.

### 📊 Examples
- Sales
- Revenue
- Quantity
- Orders
- Profit indicators

### 🧠 SQL Concepts Used
- SUM()
- AVG()
- COUNT()
- MIN()
- MAX()

---

## 4️⃣ Magnitude Analysis
Analyzing business performance magnitude across dimensions.

### 📈 Examples
- Total sales by country
- Revenue by product category
- Customer spending behavior
- Order volume analysis

### 🧠 SQL Concepts Used
- Aggregations
- GROUP BY analysis
- Business KPI calculations

---

## 5️⃣ Ranking Analysis
Ranking entities based on business performance.

### 🏆 Examples
- Top customers by revenue
- Best-selling products
- Highest-performing categories
- Customer order rankings

### 🧠 SQL Concepts Used
- ORDER BY
- TOP
- RANK()
- DENSE_RANK()
- ROW_NUMBER()

---

## 6️⃣ Top & Bottom N Analysis
Identifying highest and lowest performers within the business.

### 📉 Examples
- Top 5 customers
- Bottom 5 customers
- Top-selling products
- Lowest revenue categories

### 🧠 SQL Concepts Used
- TOP N queries
- ASC / DESC sorting
- Ranking functions
- Analytical filtering
